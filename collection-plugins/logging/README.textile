h1. Logging Plugin for Spring Insight

h2. Compatibility

This plugin is compatible with the 1.8+ versions of insight. It may
be compatible with 1.5+ (not tested).

h2. Installation

You may install this plugin by copying the plugin jar to
<code>
  insight/collection-plugins
</code>

For both the Dashboard and Insight Agent. You should see the
plugin listed under the administration panel.
