h3. Struts2 Plugin for Spring Insight

Provides data for following aspects of the Struts2 framework:
Start Proxy Execution, Custom Interceptions, Actions and Validation, Execution Result

h2. Compatibility

This plugin is compatible with the 1.9+ versions of insight. It may
be compatible with 1.5+ ( not tested ).

h2. Installation

This is a core plugin of Insight 1.9+ but earlier users may
install this plugin by copying the plugin jar to
<code>
  insight/collection-plugins
</code>

For both the Dashboard and Insight Agent. You should see the
plugin listed under the administration panel.
